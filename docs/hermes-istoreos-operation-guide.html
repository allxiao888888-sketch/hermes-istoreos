1 下载 Hermes 安装包

➜ 从 GitHub Releases 获取

  1. 访问 https://github.com/allxiao888888-sketch/hermes-istoreos/releases
  2. 下载最新版本的 luci-app-hermes.tar.gz
  3. 或者直接克隆仓库到本地：git clone https://github.com/allxiao888888-sketch/hermes-istoreos.git
  4. 在本地运行 chmod +x build.sh && ./build.sh 构建安装包
✅ 构建完成后，安装包在 dist/luci-app-hermes.tar.gz

2 iStoreOS 商店手动安装

➜ 方式 A：通过商店 GUI 安装（推荐）

iStoreOS Web 管理界面 — 192.168.1.1
iStore 商店 → 手动安装 ⭐
  1. 打开浏览器，访问 http://192.168.1.1（或你的路由器 IP）
  2. 输入管理员账号密码登录 iStoreOS
  3. 在左侧菜单找到 「iStore 商店」 进入
  4. 点击右上角 「手动安装」 按钮
  5. 选择 luci-app-hermes.tar.gz 文件并上传
  6. 等待安装完成，状态显示 ✅ 安装成功

➜ 方式 B：SSH 命令行安装

💡 适合熟悉命令行的用户，或者商店 GUI 无法正常工作时使用
  1. 将安装包复制到路由器：scp dist/luci-app-hermes.tar.gz root@192.168.1.1:/tmp/
  2. SSH 登录路由器：ssh root@192.168.1.1
  3. 解压安装包：tar xzf /tmp/luci-app-hermes.tar.gz -C /
  4. 重启 Web 服务：/etc/init.d/uhttpd restart
✅ 安装完成！刷新浏览器即可看到效果

➜ 安装后：首次配置

⚠️ Hermes 需要 Python3 来运行后端 API。如果你的路由器尚未安装 Python3，请先执行： 
opkg update && opkg install python3
然后启动 Hermes 后台服务： 
/etc/init.d/hermes-router-api enable
/etc/init.d/hermes-router-api start
✅ 服务启动后，就可以在 iStoreOS 中看到 Hermes Agent 菜单了！

7 典型使用场景

📦 场景 1：安装软件

在 AI 聊天中输入：帮我安装 frpc 内网穿透客户端
AI 会自动执行 opkg install frpc，完成后告诉你结果

🔄 场景 2：重启服务

在 AI 聊天中输入：dnsmasq 好像出问题了，帮我重启一下
AI 会执行 /etc/init.d/dnsmasq restart 并反馈结果

🔍 场景 3：查看网络状态

在 AI 聊天中输入：看看哪个设备在用我的 Wi-Fi
AI 会通过 API 获取 DHCP 租约列表和 Wi-Fi 客户端信息

⚡ 场景 4：系统诊断

在 AI 聊天中输入：路由器变慢了，帮我诊断一下
AI 会检查 CPU 负载、内存占用、网络接口错误计数、dmesg 日志

🔐 场景 5：防火墙配置

在 AI 聊天中输入：开放 8080 端口对外网访问
AI 会通过 uci 或 iptables 添加防火墙规则

⚙️ 后端 API 服务

  • • 零外部依赖（仅 Python 标准库）
  • • 监听 127.0.0.1:9120，对外安全
  • • 通过 /etc/init.d/hermes-router-api 管理
  • • 开机自启 / 停止 / 重启
  • • 配置存储在 /etc/config/hermes.json

🔒 安全机制

  • • API 端口绑定 127.0.0.1，仅本机访问
  • • /api/exec 命令白名单（禁止 rm -rf / 等）
  • • 可选的本地 API 密钥认证
  • • LLM API 密钥加密存储
  • • 所有操作日志记录

🌐 支持的 LLM

  • • OpenRouter（推荐，免费可用）
  • • OpenAI (GPT-4o / GPT-4o-mini)
  • • DeepSeek (V3 / R1)
  • • 任何 OpenAI 兼容 API
  • • 自定义 API 地址和模型名
