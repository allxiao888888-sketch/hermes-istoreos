## 环境要求与安装

### 路由器最低配置

| 项目 | 要求 |
|------|------|
| **系统** | iStoreOS / OpenWRT 21.02+ |
| **CPU** | 任意架构 (x86, ARM, MIPS) |
| **内存** | ≥ 256MB（推荐 512MB+） |
| **存储** | ≥ 64MB 空闲空间 |
| **Python** | Python3（通过 `opkg install python3` 安装） |
| **网络** | 路由器能访问 LLM API（OpenRouter 等） |

### 安装步骤

```
┌─────────────────────────────────────────────┐
│          iStoreOS Web 管理界面               │
│                                              │
│  ┌──────────────────────────────────────┐   │
│  │ ① 打开「iStore 商店」                │   │
│  │ ② 点击右上角「手动安装」             │   │
│  │ ③ 上传 luci-app-hermes.tar.gz       │   │
│  │ ④ 点击「安装」                       │   │
│  │ ⑤ 刷新页面                          │   │
│  └──────────────────────────────────────┘   │
│                                              │
│  安装后在「服务」菜单中找到「Hermes Agent」  │
└─────────────────────────────────────────────┘
```

### 首次启动 API 服务（SSH）

```bash
# 安装 Python3
opkg update && opkg install python3

# 启动 Hermes API 服务
/etc/init.d/hermes-router-api enable
/etc/init.d/hermes-router-api start

# 检查状态
/etc/init.d/hermes-router-api status
# 查看日志
logread | grep hermes-router-api
```

### 推荐提供商/模型

| 提供商 | 推荐模型 | 费用 | 设置要点 |
|--------|---------|------|---------|
| **OpenRouter** 🥇 | `google/gemini-2.0-flash-lite-2-05` | 免费 | 注册 OpenRouter 获取 Key |
| **OpenAI** | `gpt-4o-mini` | 低费用 | 需要 OpenAI API Key |
| **DeepSeek** | `deepseek-chat` | 极低 | 需要 DeepSeek API Key |
| **自定义** | 任意 OpenAI 兼容 | 自定 | 填写 API 地址和 Key |

### 免费 OpenRouter 注册

```
1. 访问 https://openrouter.ai/keys
2. 注册账号（支持 Google/GitHub）
3. 创建 API Key
4. 复制 Key 填入 Hermes 设置
5. 免费额度包含 Gemini 2.0 Flash 等模型
```

## 故障排查

### API 服务无法启动

```bash
# 1. 检查 Python3 是否安装
python3 --version

# 2. 手动启动查看错误
python3 /usr/libexec/hermes-router-api/server.py --port 9120

# 3. 检查端口占用
netstat -an | grep 9120

# 4. 查看系统日志
logread | grep hermes
```

### 常见问题

| 症状 | 原因 | 解决 |
|------|------|------|
| 仪表盘显示"未连接" | API 服务未启动 | `python3 /usr/libexec/hermes-router-api/server.py &` |
| AI 聊天报"API 错误" | LLM 密钥未配置/无效 | 进入设置填写正确的 API Key |
| 软件包列表空白 | opkg 数据库问题 | 点击"更新列表"按钮 |
| 服务操作无反应 | 权限不足 | 确认以 root 运行 |
| 页面加载慢 | 路由器性能不足 | 降低刷新间隔（设置中调整） |

### SSH 快速诊断

```bash
# 查看 API 服务状态
/etc/init.d/hermes-router-api status

# 重启服务
/etc/init.d/hermes-router-api restart

# 直接手动启动（前台调试）
python3 /usr/libexec/hermes-router-api/server.py --port 9120 --debug
```
